* Introduction
  This document reveals the plugins used in custom moodle.

* Steps to install Plugin 
  + Download the plugin to be installed.
  + Login as admin.
  + Site administration --> Plugins --> Install Plugins
  + Upload the Plugin zip file
  + Select the plugin type (if known) else ignore.
  + Click Install 
  + Moodle will guide you through till it upgrades the database.
